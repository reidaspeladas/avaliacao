    O que é Git e GitHub?

    O Git é um sistema de controle de versão distribuído amplamente usado para rastrear as mudanças em arquivos de código-fonte durante o desenvolvimento de software. O GitHub, por outro lado, é uma plataforma de hospedagem de código que utiliza o Git como sistema de controle de versão. O GitHub permite que você compartilhe seu código, colabore com outros desenvolvedores e gerencie projetos de forma eficiente.

    Configurando Chaves SSH

    Para configurar chaves SSH para autenticação segura com o GitHub, siga estas etapas:

    1. Gerar um par de chaves SSH no seu computador:
      2. ssh-keygen -t rsa -b 4096 -C "seu-email@example.com"
    2. Adicione sua chave SSH ao agente SSH:
      3. ssh-add ~/.ssh/id_rsa
    3. Copie sua chave SSH pública para a área de transferência:
      4. clip < ~/.ssh/id_rsa.pub
    4. Adicione a chave SSH pública ao GitHub nas configurações da sua conta.

    Criando e Clonando um Repositório

    Para criar um novo repositório no GitHub e cloná-lo para o seu computador, siga estas etapas:

    1. Faça login na sua conta do GitHub.
    2. Clique no botão "New" para criar um novo repositório.
    3. Preencha as informações do repositório, como nome e descrição.
    4. Clique em "Create Repository".
    5. Para clonar o repositório para o seu computador, use o comando Git:
      6. git clone git@github.com:seu-usuario/seu-repositorio.git
